# 🏎️ Supercar Showroom Management System (SQL)

## 📌 Giới thiệu
Dự án quản lý showroom siêu xe tập trung vào trải nghiệm khách hàng VIP và tối ưu hóa vận hành kho hàng.

## 🛠️ Công nghệ sử dụng
- Hệ quản trị CSDL: Microsoft SQL Server (SSMS)
- Sơ đồ ERD: Draw.io

## 📊 Các tính năng chính
- Quản lý danh mục siêu xe độc bản (Unique VIN).
- Theo dõi lịch sử giao dịch và sở hữu của khách hàng VVIP.
- Báo cáo doanh thu và hiệu suất dịch vụ.

## 📂 Cấu trúc thư mục
- `/Database`: Chứa file `.sql` cài đặt.
- `/Documents`: Sơ đồ ERD và Slide thuyết trình.
